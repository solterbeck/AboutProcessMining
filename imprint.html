About the Library

App Information

This web application was developed using HTML, JavaScript (including jQuery), and Bootstrap. It is hosted as a static site on GitHub Pages at https://solterbeck.github.io/AboutProcessMining/. Being a static website, it requires no server-side processing. Data is stored in a public GitHub repository.

Please note that this application is "Under Construction" and serves as a learning prototype. Some features and improvements are actively being developed.

Data Privacy and Protection

Since the application is hosted on GitHub Pages, the repository "About Process Mining" is public, making data stored within accessible to anyone. Personal information (e.g., names, email addresses, or passwords) is not collected, processed, or stored by this application. Data is only available for viewing as it is provided within the public repository.

Users are encouraged to refrain from submitting sensitive information via links or external sources associated with this application, as GitHub does not provide additional security for public repositories.
